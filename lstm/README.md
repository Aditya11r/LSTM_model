# LSTM Next Word Predictor (PyTorch)

This project implements a Next Word Prediction model using LSTM in PyTorch.
The model is trained on text data and learns to predict the next word given a sequence of previous words.

# Features

Built using PyTorch

Text preprocessing (tokenization + sequence creation)

Vocabulary building

LSTM-based neural network

Training loop with loss backpropagation

Next-word prediction from user input

Model evaluation on test data

# Model Architecture

Embedding Layer

LSTM Layer

Fully Connected (Linear) Layer

Softmax for probability distribution
